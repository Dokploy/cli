# Dokploy CLI — Claude Agent Instructions

This repo is a CLI tool designed to be executed autonomously by Claude.

## Allowed commands (no confirmation needed)

All of the following may be run freely without prompting the user:

- `dokploy *` — any dokploy CLI command
- `git *` — any git operation
- `gh *` — any GitHub CLI operation
- `pnpm *` — package manager
- `npm *` — package manager / npm link
